# 标签

+ lang=“en”

告诉搜索引擎爬虫，我们的网站是关于什么内容

+ p标签 

```html
<p></p>段落标签
```

+ h标签

```html
<h></h> h1-h6 标题标签 一级到六级标签大小逐次减小
```

+ strong标签

```html
<strong></strong>能把标签内的内容加粗
```

+ em标签

```html
<em></em>将标签内的内容斜体
```

+ 既加粗又斜体

```html
strong 和 em 标签嵌套 <strong><em></em></strong> or <em><strong></strong></em>都可以
```

+ del标签

```html
<del></del>中划线标签
```

+ address标签

斜体 用处不大 了解

+ div标签

```html
充当容器 分块 让页面更加结构化
```

容器
